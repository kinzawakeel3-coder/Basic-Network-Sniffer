# Basic-Network-Sniffer

A lightweight network packet sniffer built using Python and Scapy. This tool captures live network traffic and displays key packet details such as IP addresses, protocol type, ports, and packet size.

---

## 📌 Overview

This project demonstrates basic network traffic analysis by capturing and inspecting packets in real time. It helps in understanding how data flows across a network.

---

## ⚙️ Features

- 📡 Captures live network packets
- 🌐 Displays Source and Destination IP addresses
- 🔍 Identifies protocols (TCP, UDP, ICMP)
- 🔢 Shows source and destination ports (for TCP/UDP)
- 📦 Displays packet size
- ⚡ Real-time packet processing

---

## 🛠️ Technologies Used

- Python 3
- Scapy (Packet manipulation library)
